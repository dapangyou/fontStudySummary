HTML 标签
```js
    1.段落标签和换行标签是有区别的,段落和段落之间是有空格的。段落占满一行才会换行,
    <br>换行是强制换行 中间不会有大的空格
    2.文本格式化标签:
        文字加粗标签:    <b>  <strong>:语义化会更加强烈
        文字斜体标签：   <i>  <em> :语义化更加强烈
        删除线:         <s>  <delete>: 语义化更加强烈
        下划线:         <u>  <ins> 
    3.div和span
        div和span都是盒子
        div:会独占一行
        span: 不会独占一行  会横向排列
    4.a标签
        target有两个值  
            _blank: 重新打开一个浏览器窗口打开
            _self:  在当前浏览器窗口中打开
        href的值不同的几种表现形式
        (1)  外部链接：http://....会跳转到网络网页地址
        (2)  内部链接：/src/...   相对于当前地址的相对路径
        (3)  锚点链接： #  不知道要跳转到哪里的时候暂时可以放#
        (4)  下载链接: /.zip /.exe  路径是.zip / .exe的路径 可下载
    5.锚点链接
        href中设置其他标签的id名称
        <h1 id="title"><h1>
        <a href="#title">  
    6.表格的作用是用来展示数据的
    表格标签
        cellpadding: 表格内容和表格边框线的距离
        cellspacing: 表格单元格与单元格之间的距离
    7.列表
        列表的作用就是用来布局的  使得页面干净整洁
        有序标签:
            <ol><li></li></ol>
        无序标签
            <ul><li></li></ul>
        自定义标签
            <dl>
                <dt></dt>
                <dd></dd>
                <dd></dd>
            <dl>
```

CSS
```js
    链接伪类选择器
        a:link: 从未选择过的链接
        a:visited: 选择已访问过的链接
        a:hover:  鼠标悬浮的链接
        a:active: 激活的链接
    注意: 为了确保生效 一定要有顺序  LVHA

    :focus 伪类选择器
        针对于input的伪类选择器  
    input:focus {
        background-color: red
    }

    元素显示模式
    元素的显示模式一般分为块级元素和行级元素
    块级元素
        <div> <h1>~<h6> <p> <ol> <ul> <li>
    行级元素
        <span> <a>
    注意: 文字类的元素中不能放块级元素 比如 <p>元素是主要用于存放文字类
          <p>元素中不能放<div>等块级元素
    
    块元素的特点:
        1.独占一行
        2.有宽高  外边距及内边距都可以控制
        3.宽度默认是父级容器的宽度的100%
        4.是一个容器或盒子  可以放块级或者行级元素
    行级元素的特点:
        1.相邻的行内元素可以在一行 一行可以显示多个
        2.高度、宽度设置是无效的
        3.默认宽度就是本身的宽度
        4.行内元素只能容纳文本或其他行内元素
```
