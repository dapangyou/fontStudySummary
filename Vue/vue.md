指令

```js
    v-once指令:有v-once指令只加载data中的数据一次 之后不再加载
               即使data中的数据改变了 界面上也不会有改变
    v-html指令:解析标签
    v-text指令: 在页面展示data中的数据
                这个指令和插值表达式的功能一样
    v-text指令和插值表达式的区别?
                v-text指令没有插值表达式好用 如果后面追加一个值
                那么v-text中会覆盖其他的值 插值表达式就会追加值
    v-pre指令:  加上该指令 就不会解析插值表达式了  插值表达式会原封不动的显示到页面上
    v-cloak指令: vue还没有被解析的时候,会直接显示插值表达式的内容,然后等到vue解析之后才会显示数据内容
                 加上v-cloak 然后style添加属性样式display: none 可不显示 看起来更加直观
```
